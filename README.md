# Miruthul Kumar M Portfolio

A premium frontend-only portfolio built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Run Locally

```bash
npm install
npm run dev
```

## Structure

- `public/profile.jpg` - hero profile image asset
- `public/Miruthul_Kumar_Resume.pdf` - downloadable resume
- `src/data/portfolioData.js` - all portfolio content
- `src/components/` - reusable portfolio sections

## Notes

- The contact form is UI-only and shows a success state on submit.
- Replace the placeholder public assets with final personal media whenever needed.
